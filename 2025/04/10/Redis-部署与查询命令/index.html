Redis支持的数据结构与常用查询

通用命令

KEYS 查看符合模板的所有key

DEL 删除一个指定的key

EXISTS 判断key是否存在

EXPIRE 给一个key设置有效期

TYPE 查看key对应值的数据类型

RENAME 重命名key

字符串String

String类型支持三种数据格式：

  • string 字符串
  • int 整数
  • float 浮点数

基本操作

SET/GET 设置/获取字符串值

MSET/MGET 批量设置/获取多个键值对

字符串操作

STRLEN 获取字符串长度

GETRANGE 截取字符串

SETRANGE 替换字符串的部分内容

APPEND 向字符串末尾追加内容

数值操作

INCR/DECR 自增/自减

INCRBY/DECRBY 按指定步长自增/自减

INCRBYFLOAT 浮点数自增

条件设置

SETNX 仅在键不存在时设置值

SETEX 设置值并指定过期时间

Hash 哈希表（散列表）

Hash类型用于存储对象，相当于一个小型的key-value数据库。

基本操作

HSET/HGET 设置/获取哈希表中的字段

HMSET/HMGET 批量设置/获取多个字段

HGETALL 获取哈希表中所有字段和值

字段操作

HKEYS/HVALS 获取所有字段名/字段值

HDEL 删除指定字段

HEXISTS 判断字段是否存在

HLEN 获取字段数量

HSTRLEN 获取字段值的长度

数值操作和条件设置

HINCRBY 对字段值进行数值自增

HSETNX 仅在字段不存在时设置值

HRANDFIELD 随机获取字段

List 列表（双向链表）

List是一个有序的字符串列表，支持从两端进行插入和删除操作。

插入操作

LPUSH/RPUSH 从左/右侧插入元素

LPUSHX/RPUSHX 仅在列表存在时从左/右侧插入

LINSERT 在指定元素前/后插入新元素

查询操作

LRANGE 查询指定范围的元素

LINDEX 获取指定位置的元素

LPOS 查找元素的位置

LLEN 获取列表长度

删除和弹出操作

LPOP/RPOP 从左/右侧弹出元素

LREM 删除指定数量的指定元素

LTRIM 保留指定范围的元素，删除其他元素

阻塞操作

BLPOP/BRPOP 阻塞式弹出元素

Set 集合

Set是一个无序的、不重复的字符串集合。

基本操作

SADD 向集合添加元素

SMEMBERS 获取集合的所有成员

SREM 从集合中移除元素

SPOP 随机移除并返回元素

查询操作

SCARD 获取集合元素个数

SISMEMBER 判断元素是否存在于集合中

SRANDMEMBER 随机获取元素（不删除）

集合运算

SINTER 求多个集合的交集

SDIFF 求多个集合的差集

SUNION 求多个集合的并集

SMOVE 将元素从一个集合移动到另一个集合

Sorted Set 有序集合

Sorted Set是一个有序的、不重复的字符串集合，每个元素都关联一个分数用于排序。

基本操作

ZADD 添加元素及其分数

ZRANGE 按排名范围获取元素

ZRANGEBYSCORE 按分数范围获取元素

ZRANGEBYLEX 按字典序范围获取元素

查询操作

ZRANK/ZREVRANK 获取元素的排名

ZSCORE 获取元素的分数

ZMSCORE 获取多个元素的分数

ZCARD 获取集合中元素的数量

ZCOUNT 统计分数范围内的元素数量

修改操作

ZINCRBY 增加元素的分数

ZREM 删除元素

ZREMRANGEBYSCORE 按分数范围删除元素

ZREMRANGEBYLEX 按字典序范围删除元素

ZREMRANGEBYRANK 按排名范围删除元素

ZPOPMAX/ZPOPMIN 弹出分数最高/最低的元素

集合运算

ZDIFF 计算多个有序集合的差集

ZINTER 计算多个有序集合的交集

ZUNION 计算多个有序集合的并集
